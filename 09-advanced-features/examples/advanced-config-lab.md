# Advanced Config Lab

> 📌 **This is a hands-on companion to [Module 09: Advanced Features](../README.md).** Use `~/opencode-practice` for the guided path, then transfer to your own projects.

## Objectives

- Inspect and tune advanced permission/config settings
- Validate formatter/LSP behavior on real files
- Practice reversible changes with checkpoints and undo/redo

## Prerequisites

```bash
cd ~/opencode-practice
opencode
```

## Exercises

### Exercise 1: Permission Audit

Prompt:

```
Audit this project's permission configuration and suggest one safer default
```

### Exercise 2: Formatter/LSP Check

Prompt:

```
Make one small code style change and verify formatter and diagnostics behavior
```

### Exercise 3: Checkpoint and Recovery

Prompt:

```
Create a small reversible change, then use undo/redo to verify recovery flow
```

### Exercise 4 (Transfer): Advanced Settings in Your Project

Apply one permission hardening and one formatter/LSP adjustment in a personal repo.

## Verification Checklist

- [ ] I can explain permission changes and their impact
- [ ] I verified formatter/LSP behavior after edits
- [ ] I can safely recover from a bad edit using checkpoint workflows
